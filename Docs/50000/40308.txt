<HEADLINE>
UEFA'dan Türk hakemlerine görev... 
</HEADLINE>
<TEXT>

     UEFA Hakem Komisyonu, FIFA kokartlı 3 Türk hakeme, UEFA Kupası maçlarında görev verdi. 
     Futbol Federasyonu'ndan yapılan açıklamaya göre, yardımcıları ve 4. hakemleri Futbol Federasyonu Merkez Hakem Kurulu tarafından belirlenen Türk hakemlerin 13 Eylül'de yönetecekleri karşılaşmalar şöyle 
     Erol Ersoy NK Olimpija Ljubliana-Brondby IF 
     Yardımcıları Adnan Ayçiçek, Aykut Gümülü, 4. hakem Selçuk Dereli 
     Orhan Erdemir Ipswich Town-FC Torpedo Moskova 
     Yardımcıları Baki Tuncay Akkın, Kenan Altınsaat, 4. hakem Ferhat Gündoğdu 
     Metin Tokat Real Zaragoza-Silkeborg IF Yardımcıları Cengiz Akyüz, Sürhat Müniroğlu, 4. hakem Murat Ilgaz
