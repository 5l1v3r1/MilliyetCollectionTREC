<HEADLINE>
Microsoft, Windows kaynak kodunu devletlere açıyor... 
</HEADLINE>
<TEXT>

     Microsoft, Devlet Güvenlik Programı DPG ile devletlere Windows kaynak kodunu açacağını duyurdu. 
     Microsoft, hükümetlerin ve belli başlı devlet kurumlarının bilgi teknolojisi yatırımlarının güvenlik açısından diğer tüketicilerden daha fazla tehdit altında bulunması nedeniyle geliştirdiği Devlet Güvenlik Programı'nı DGP açıkladı. 
     Microsoft'tan yapılan açıklamaya göre, şirketin dünya çapında uygulayacağı Devlet Güvenlik Programı'na göre, ulusal güvenlik konuları ve vatandaşlara ait kişisel bilgilerin saklanması konusundaki hassasiyet nedeniyle devletler, Windows kaynak koduna erişim iznine sahip olacaklar. Programa göre devletler ayrıca, Windows platformunu ileri güvenlik özellikleriyle donatmak için gerekli olan teknik bilgileri Microsoft'tan alabilecekler. 
     DGP anlaşmasının şu an Rusya ve Türkiye'nin de üyesi bulunduğu NATO tarafından imzalandığı, 20 ülkeyle de görüşmelerin sürdüğü bildirildi. 
     DGP, Microsoft'un devletlerin ve dünyadaki uluslararası organizasyonların güvenlik ihtiyaçlarına yanıt vermek adına yürürlüğe koyduğu programın ayrılmaz bir parçasını oluşturuyor. 
     Program, hükümetlerin ülkelerine özel güvenlik ihtiyaçlarını karşılamak üzere geliştirildi. Microsoft'un hiçbir ücret talep etmeden sunduğu hizmet, kod gözden geçirme araçlarıyla Windows koduna erişimi kapsıyor. Bazı lisans sınırlamalarına tabi olan program aynı zamanda Windows Platformu ile ilgili teknik bilginin paylaşılmasını içeriyor. 
     Program ayrıca, teknik bilgi paylaşımı nedeniyle Microsoft'un bilgi teknolojileri güvenliği konusundaki uzman ekipleri ile hükümet yetkililerinin birlikte çalışmalarını sağlayacak.
