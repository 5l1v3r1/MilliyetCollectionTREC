<HEADLINE>
Türk müteahhitlerine ABD'den Irak daveti 
</HEADLINE>
<TEXT>

   ABD, Irak'taki 18.7 milyar dolarlık ihaleler için Türk şirketlere 'Hızla harekete geçin' mesajı gönderdi 
      
   ABD, Irak'ın yeniden inşası için ayırdığı 18.7 milyar dolar tutarındaki ihalelere katılacak ülkelerin listesini hazırlamaya başladı. 
   Liste henüz belli olmadı ama ABD'nin ekonomik işlerden sorumlu Dışişleri Bakan Yardımcısı Alan Larson "Ben Irak'la ilgilenen bir Türk şirketinin yöneticisi olsam, ihaleler için hızla çalışmaya başlardım" dedi. Türk şirketleri, şu internet adreslerinden bilgi alabilecek www.cpairaq.org, http www.rebuildingiraq.net, http www.iraqcoalition.org business, www.export.gov iraq, http kirkukbusinesscenter.org, http iraqibusinesscenter.org, http portaliraq.com, http www.usaid.gov iraq ve http www.iraqdaily.com.
