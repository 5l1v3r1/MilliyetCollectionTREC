<HEADLINE>
Oyakbank 'Kullan-at' şifreye geçti 
</HEADLINE>
<TEXT>


EKONOMİ SERVİSİ 

Oyak Bank, bireysel internet bankacılığına giriş java uyumlu cep telefonları aracılığıyla tek seferlik şifreler üreten "Kullan-at" güvenlik sistemini uygulamaya başladı. Kullanımı ücretsiz olan sistem, internet bankacılığı işlemleri için, java uyumlu cep telefonu aracılığıyla her defasında otomatik olarak tek kullanımlık şifreler üretiyor. Şifreler kullanıldıktan sonra kendiliğinden geçersiz hale geliyor.
